Game "Find the Buried Treasure".
The aim of thi game is to find the hidden treasure.The web page displays a treasure map. Inside the map, the program will pick a single pixel location, which represents where the hidden treasure is buried. Every time the player clicks the map, the web page will tell them how close to the treasure they are. When they click the location of the treasure (or very close to it), the game congratulates them on finding the treasure and says how many clicks it took to find it.  

Used technologies: JS, jQuery
